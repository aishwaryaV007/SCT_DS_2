# 🚢 Titanic Dataset - Data Cleaning and Exploratory Data Analysis (EDA)

## 📌 Task Overview

This repository contains the submission for **SkillCraft Technology Data Science Internship - Task 02**.

The objective of this task is to perform **Data Cleaning** and **Exploratory Data Analysis (EDA)** on the Titanic dataset to identify patterns, trends, and relationships among variables that may have influenced passenger survival.

---

## 🎯 Task Objective

* Clean and preprocess the Titanic dataset.
* Handle missing values and inconsistencies.
* Explore passenger demographics and travel information.
* Analyze factors affecting passenger survival.
* Visualize relationships between variables using statistical plots.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

---

## 📂 Dataset

Titanic Dataset from Kaggle: *Titanic - Machine Learning from Disaster*

---

## 📚 Conclusion

Through this task, data cleaning techniques and exploratory data analysis methods were applied to the Titanic dataset. The analysis revealed important patterns related to passenger survival, particularly with respect to gender, passenger class, age, and fare. The cleaned dataset and visualizations provide a solid foundation for future predictive modeling tasks.
